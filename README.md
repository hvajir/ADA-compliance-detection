# ADA Compliance Detection System

AI-powered accessibility auditing for retail spaces using YOLOv8 computer vision.

## MIT 6.S191 Deep Learning Project
Author: Hardi Vajir
Date: January 2026

## Overview
Automated detection of ADA (Americans with Disabilities Act) compliance violations using:
- YOLOv8 object detection
- Computer vision analysis
- Automated report generation

## Quick Start
```bash
# Install dependencies
pip install -r requirements.txt

# Run demo
python3 demo.py test_images/your_image.jpg
```

## Features
- Detects doors, parking spaces, ramps, pathways
- Identifies potential ADA violations
- Generates annotated images
- Produces detailed JSON reports
